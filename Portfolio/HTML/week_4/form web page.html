Form





HTML Form

A section of a document containing normal content, markup, special elements called controls.

HTML Form Action

specifies where to send the form-data when a form is submitted.

HTML Form Method ( GET & POST )

The HTML method specifies how to send form data. The form data is sent to a specific page from the action attribute. The GET method sends information up in the URL bar while POST method hides it from the bar. The POST method is more secure than the GET method.

HTML Fieldset

The fieldset tag in HTML5 is used to make a group of related elements in a form.

HTML Legend

In web development, the legend element is used in HTML forms to provide a caption.

HTML input tag

The HTML input tag defines the field where the user can enter data.

HTML attributes

Attributes help modify and control HTML elements.

HTML checkboxes

used to let a user select one or more options of a limited number of choices.

HTML radio buttons

The HTML " < input type="radios" > defines a radio button, grouped by sharing the same attribute.

HTML textarea

HTML textarea is an element that is also used often as a form to collect user inputs and comments or reviews.

HTML button

A clickable button, the user can use it to submit stuff.

HTML label

The label element is used to associate a text label with a form input field.
