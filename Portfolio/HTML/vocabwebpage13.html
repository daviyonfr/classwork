vocabulary

Uniform Resource Locator (URL) the address of a unique resource on the internet.
Firewall a system that monitors and controls network traffic, allowing or blocking data packets based on a set of security rules, acting as a barrier between a trusted internal network and untrusted external networks
Transmission Control Protocol-Internet Protocol (TCP-IP) a suite of communication protocols that enables data transmission between computers on a network, including the internet, by breaking data into packets, addressing, routing, and ensuring reliable delivery
Network Protocols a markup language used to structure webpages
Computer Network a system where two or more computers or devices are interconnected to share data, resources, and communicate, often using protocols like HTTP and TCP/IP, and where HTML is used to structure web content
IPv4 Internet Protocol version 4, a protocol that assigns unique numerical addresses to devices on a network, enabling communication over the internet
IPv6 a 128-bit alphanumeric value that identifies an endpoint device in an Internet Protocol Version 6 (IPv6) network.
Domain Name Services (DNS) In the context of the internet and web development, Domain Name Service (DNS) is a system that translates human-readable domain names (like example.com) into IP addresses (like 192.0.2.1), which computers use to communicate
Hyper-text transfer protocol (HTTP/HTTPS) the main way web browsers and servers communicate to share information on the internet
File Transfer Protocol a standard network protocol used for the transfer of files from one host to another over a TCP-based network, such as the Internet.
Cloud Computing e, databases, networking, and software, over the internet, allowing users to access and utilize these services without needing to manage or maintain the underlying infrastructure
OSI Model a conceptual framework that standardizes network communication by dividing it into seven layers, each responsible for specific tasks, helping to understand and troubleshoot network issues
