Week 14 Vocab

Wireframe A visual diagram that outlines the skeletal framework of a website, app, or other digital product.
User story A user story is an informal general explanation of software feature written from the respective of the end use.
mood board Mood board shows UX design and makes people feel emotions.
LATCH The only 5 ways top organize information are location, alphabet, time category and hierarchy.
sans-serif fonts serif fonts Serif fonts have small strokes or flourishes (serifs) at the ends of letter strokes, while sans-serif fonts lack these, resulting in a cleaner, more minimalist look.
Usability how easily and effectively a user can interact with and accomplish tasks using a product, system, or website, focusing on efficiency, satisfaction, and little stress.
Sitemaps for Google (XML) Sitemaps are purely for search engines. HTML sitemaps were also recommended for search engine optimization ( SEO ) purposes.
Sitemaps visual A sitemap is a flowchart that shows how different pages and content on a website are connected. It helps designers and developers plan the layout and structure of a website before building it.
story board A visual representation of a film sequence and breaks down the action into individual panels.
Static and Variable web fonts Static fonts are fixed and don't offer a wide range of styles, while web fonts, loaded from the internet, allow for more flexible typography and wider font choices on websites.
Card sort Card sorting can be for learning how people function individually or can be used for the creators of a website.
form tag with action and method attributes The method attributes specifies how to send form data. The form data is sent to the page specified in the action.
